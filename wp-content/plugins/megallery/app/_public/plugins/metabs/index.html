MeTabs

Divisor de conteúdo em abas

Desenvolvido por: Wallace Rio

Exemplos:

  • Aba simples
  • Aba bloqueada
  • Aba lateral
  • Aba automática (slider)
  • Aba automática (slider com callback ao finalizar)
  • Aba callback (executa javascript)
  • Aba com controle externo
  • Aba multipla
  • Aba nova animação
  • Aba com controle via hash
  • Carrosel logos

USAGE:

Requesito: JQuery é requisito para este plugin

Inclua o CSS e o JS no HEAD, depois apenas inclua a marcação HTML

<link rel="stylesheet" href="metabs/metabs.css"'>
<script type="text/javascript" src='metabs/metabs.js'></script'>

1. Crie uma tag DIV com o parametro data-metabs


2. Crie outra tag DIV com o parametro data-rel='tabContainer', e dentro desta div crie as abas com o parametro data-rel='metabs_tab', e com o parametro data-for com um valor único de identificação para o conteúdo da aba, para definir qual aba deverá ficar ativa ao iniciar utilize o parametro data-state='active'


3. Para o conteúdo crie as DIV's relacionadas as abas com os parametros data-rel='metabs_content', id="formContent1" para cada identificação
